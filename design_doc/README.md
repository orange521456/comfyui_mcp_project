## 已确认的关键决策

| 决策项 | 结论 |
|--------|------|
| 工作流创建策略 | 模板驱动 + 动态构建结合 |
| 通信方式 | 先 REST，后期升级 WebSocket |
| MCP Server 语言 | Python（FastMCP） |
| 工作流表达 | 中间表示层（IR） |
| 模板存储 | 内置模板 + 用户自定义模板文件夹 |
| 参数暴露 | 常用参数暴露，高级参数保留默认值但可选覆盖 |
| 图片返回 | 缩略图 base64 + 原图路径 |
| 原子操作 | 中间表示层覆盖 80% 场景，原子操作用于兜底 |
| ComfyUI 发现 | 启动时自动检测，不可达不阻塞启动 |
| 错误处理 | 统一 `{"success": false, "error": "...", "error_code": "..."}` 格式 |
| 多客户端 | 暂不支持，后续可升级 |
| 日志 | stderr + 文件（按天轮转，保留 7 天），默认 INFO 级别 |
| 只读数据 | MCP Resources 暴露模型列表、节点类型等 |
| 模板批次 | 第一批 txt2img + img2img；第二批 inpaint + upscale + animation |
| 不纳入 | 音效制作、背景音乐（不在 ComfyUI 能力范围内） |
| Python 版本 | Python 3.10.0 |
| 包管理工具 | pip（`requirements.txt`） |
| 测试策略 | 单元测试，初期不 mock ComfyUI（手工测试） |
| MCP Prompts | 不做，现有 Tools + Resources 已够用 |
| 打包分发 | 直接运行（`python src/server.py`），不做 pip 发布或 Docker |
